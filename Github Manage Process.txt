First time use:
1. Use cd +"direct in you computer" go into the computer directory 
2. For the first time you download the github, use: github clone + SSH link
3. Connect to database: git remote add + SSH link

Everytime when you want to add something.
1. Everytime before add some file pull first: git pull 
2. After editing file use: git status 
3. git add "xx.xx".
4. git commit -m "Comment"
5. git push

Switch Branch: git checkout + "branch name"
